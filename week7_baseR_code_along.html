Lets combine all of the above

We will use numeric to create an empty vector We will use for to iterate along items of that empty vector We will use [] to access each slot and fill-in that empty vector We will use rnorm to sample values from a random normal distribution We will plot the trend

\[N_{t}=N_{t-1} + r\left(1 - \frac{N_{t-1}}{K}\right)\]

\[ r \sim norm(\mu = 1.5, \sigma = 0.8) \]

Writing the code down

# Set up mode parameters
K <- 100
nsteps <- 100
mu <- 0.4
sigma <- 1

# Set up model objects
N <- vector(mode = "numeric", length = nsteps)
N[1] <- 25

# Model it
for(t in 2:nsteps){
  r <- rnorm(n = 1, mean = mu, sd = sigma)
  N[t] <- N[t-1] + r * (1 - (N[t-1] / K))
}

# Visualize it
plot(N)

Something more

We will use matrix to create an empty matrix (rows are time, columns are runs) We will repeat the process

# Set up mode parameters
K <- 100
nsteps <- 100
mu <- 0.4
sigma <- 1

# Set up model objects -Modify
N <- matrix(data = NA, nrow = nsteps, ncol = 100)
N[1,] <- 25

# Model it
for(t in 2:nsteps){
  r <- rnorm(n = 100, mean = mu, sd = sigma)
  N[t,] <- N[t-1,] + r * (1 - (N[t-1,] / K))
}

# Visualize it
matplot(N, type = "l")
